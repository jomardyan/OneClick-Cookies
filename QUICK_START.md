## ğŸ® Using the Extension

### Manual Mode (Recommended for first-time users)

1. Click extension icon
2. Select "Manual" mode
3. When you see a cookie banner:
   - Click extension icon
   - Click "Accept All Cookies" or "Deny All Cookies"

### Auto-Accept Mode

1. Click extension icon
2. Select "Auto-Accept" mode
3. Browse normally - banners are accepted automatically

### Auto-Deny Mode

1. Click extension icon
2. Select "Auto-Deny" mode
3. Browse normally - banners are denied automatically

### Whitelist a Domain

1. Visit the website
2. Click extension icon
3. Click "Add to Whitelist"
4. Cookies will always be accepted on this site

### Blacklist a Domain

1. Visit the website
2. Click extension icon
3. Click "Add to Blacklist"
4. Extension will skip banners on this site
